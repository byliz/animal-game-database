Purpose

This website was originally made to record all video games in which animals are important to the plot or gameplay, but I realized this topic was way too broad. This website will now record only games featuring playable non-human animal characters including mythical animals like dragons.

Why?

Animals are cool. They're in so many games, but they're often not the playable character or are represented in a not very realistic manner. I love games where the animal's species is actually relevant to their abilities. I want this website to record all of these titles, no matter how obscure they may be, so that a wider audience can find them and enjoy them.

How it Works

I made this website by hand from scratch and use Github Pages to host it. You can check out the source code here. This website is a static website generated using the Jekyll static site generator. Entries must be manually added as a markup file in the _posts folder.

Contributions

The Contribute button on the navigation menu goes to an external Google Form where you can submit any suggestions for games that you think should be listed.

If you found this website useful and enjoy my work, feel free to share it with others. You can also help by donating at Ko-fi.com/lizlorena.
