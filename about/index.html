Purpose

This website was originally made to record all video games in which animals are important to the plot or gameplay, but I realized this topic was way too broad. This website will now record only games featuring dragons.

Why?

Dragons are cool. They exist in the stories of many cultures all around the world. There's a wide variety of them, and they're everywhere, yet I think there aren't enough dragons in video games. I want this website to record all of these tiles, no matter how obscure they may be, so that a wider audience can find them and enjoy them.

How it Works

I made this website by hand from scratch and use Github Pages to host it. You can check out the source code here. This website is a static website generated using the Jekyll static site generator. Entries must be manually added as a markup file in the _posts folder.

Contributions

The Contribute button on the navigation menu goes to an external Google Form where you can submit any suggestions for games that you think should be listed.

If you found this website useful and enjoy my work, feel free to share it with others. Also please consider following me on Twitter for progress updates on all my projects. You can also help by donating at PayPal.me/cozymaus.
